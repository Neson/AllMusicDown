All Music Down
==============

**BJ4**

## Development

```
bundle
```


And starts mongodb.
If you've not installed it, then installed it by running `brew install mongodb` or something.
```
mogod
```

Then starts server
```
foreman start 
```

and visit [http://localhost:5000](http://localhost:5000)

run `guard` for rspec testing, `tux` for sinatra shell playground.
